Azure Free Tier Assignment

Introduction

This project demonstrates my exploration of Microsoft Azure Free Tier services. The objective was to create an Azure account, explore the Azure Portal, customize the dashboard, review billing information, and understand Azure security best practices.

Objectives

- Create an Azure Free Tier account.
- Explore the Azure Portal.
- Customize the Azure dashboard.
- Review billing and cost management.
- Understand Azure security features.

Azure Account Creation

1. Visited the Azure Portal.
2. Signed up using my email address.
3. Verified my email address.
4. Completed phone verification using SMS.
5. Added a debit/credit card for identity verification.
6. Azure successfully created a Free Tier subscription.

Azure Portal Navigation

The Azure Portal contains a navigation menu on the left side that provides access to services and resources. The search bar at the top can be used to quickly locate services. Breadcrumbs help users know their current location within the portal.

Dashboard Customization

1. Opened the Azure Dashboard.
2. Clicked the Edit button.
3. Added useful tiles such as Resource Groups and Cost Management.
4. Renamed the dashboard to "My Josh Dashboard".
5. Saved the dashboard.

Billing and Cost Management

The Cost Management + Billing section was explored to monitor expenses. Current spending and subscription details were reviewed. A budget alert was configured to notify me when spending reaches 75% of the budget threshold.

Security Best Practices

- Use strong passwords.
- Enable Multi-Factor Authentication (MFA).
- Apply Role-Based Access Control (RBAC).
- Regularly review account activity.
Challenges Encountered
Some challenges included understanding the Azure interface and navigating billing sections. These were resolved by exploring Microsoft's documentation and portal help features.

Conclusion

This assignment provided practical experience with Microsoft Azure and improved my understanding of cloud computing concepts.
